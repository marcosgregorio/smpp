## 📚 Sobre o SMPP

O SMPP (Short Message Peer-to-Peer) é um protocolo de comunicação usado para enviar e receber mensagens curtas (SMS) entre SMSCs (Short Message Service Centers) e ESMEs (External Short Messaging Entities). O SMPP é um protocolo de comunicação de mensagens de texto que permite a troca de mensagens entre SMSCs e ESMEs.
